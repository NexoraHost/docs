### 📌 Wichtige Information zur Verfügbarkeit

**Für jeden einzelnen Server eine eigene Lizenz benötigt wird**, die nicht von uns angeboten werden.

### ❓ Warum ist das so?

GIANTS Software, die Entwickler von Farming Simulator, setzen voraus, dass **jede Instanz eines Servers separat lizenziert** wird.

> 📢 Hinweis: Ohne eine gültige Farming Simulator-Serverlizenz kann leider kein Server bereitgestellt werden.

Bei Fragen oder Unklarheiten steht dir unser Support-Team jederzeit gerne zur Verfügung\!
