> **_⚠️ Wichtiger Hinweis_**\
> Bitte beachte, dass bei einer **Neuinstallation** des Servers **alle Daten verloren gehen**. Dies bedeutet, dass alle bestehenden Daten und Konfigurationen, die auf dem Server gespeichert sind, dauerhaft gelöscht werden. Stelle sicher, dass du **alle wichtigen Daten vorher gesichert hast**, bevor du mit der Neuinstallation fortfährst.

### 🕒 Schnelle Bereitstellung nach Neuinstallation

Nach der Neuinstallation des Servers ist dein Server in der Regel innerhalb von **5-10 Minuten** wieder vollständig erreichbar. Die Installation und Bereitstellung erfolgt schnell, damit du keine wertvolle Zeit verlierst und direkt mit der Konfiguration deines Servers fortfahren kannst.

### 🌐 Verfügbare Betriebssysteme

Die folgenden Betriebssysteme stehen für die Neuinstallation zur Verfügung:

- **Debian 12**
- **Debian 11**
- **Ubuntu 20.04**
- **Ubuntu 22.04**
- **Ubuntu 24.04**
- **Fedora 37**
- **Fedora 38**
- **Windows Server 22 (DE)**
- **Windows Server 22 (EN)**

### 🔄 Prozess der Neuinstallation

1. **Wählen des Betriebssystems:** Du wählst das gewünschte Betriebssystem aus der Liste aus.
2. **Datenverlust:** Alle aktuellen Daten auf dem Server werden gelöscht. Stelle sicher, dass du vorher ein vollständiges Backup machst.
3. **Automatische Installation:** Nach Auswahl des Betriebssystems erfolgt die Neuinstallation automatisch.
4. **Verfügbarkeit:** Innerhalb von 5-10 Minuten ist der Server wieder erreichbar, und du kannst ihn über SSH oder Remote Desktop (für Windows Server) verwenden.

### 🛠️ Achtung bei der Neuinstallation

- **Alle Daten gehen verloren\!** Achte darauf, alle wichtigen Dateien zu sichern.
- **Backup erstellen:** Vor der Neuinstallation solltest du ein vollständiges Backup deiner Daten machen, wenn du diese später wiederherstellen möchtest.
